Tag Função antiga Alternativa moderna
<acronym> Exibia siglas com explicação ao passar o mouse <abbr>
<applet> Incorporava applets Java <object>, <embed>
<basefont> Definia fonte base da página CSS (font-family, font-size)
<big> Aumentava o tamanho do texto CSS (font-size)
<blink> Fazia o texto piscar (não suportado por muitos navegadores) CSS (evitar esse efeito)
<center> Centralizava conteúdo CSS (text-align: center)
<dir> Lista de diretórios <ul>
<font> Alterava cor, tamanho e tipo da fonte CSS
<frame> / <frameset> / <noframes> Layout com frames <iframe> ou CSS Grid/Flexbox
<isindex> Campo de busca (HTML 3.2) <input type="search">
<marquee> Texto rolante CSS (animação) — geralmente evitar
<menu> (HTML4 versão) Menu simples (não usado como lista) <ul> + CSS
<nobr> Impedia quebra de linha CSS (white-space: nowrap)
<s> (reintroduzido com nova semântica) Texto riscado (visual) <del> ou evitar uso visual
<strike> Texto riscado (não semântico) <del>
<tt> Texto em fonte monoespaçada (teletype) CSS (font-family: monospace)
<u> (HTML4, sem semântica) Sublinhado visual CSS (text-decoration: underline) ou <ins>

Negrito e Italico

Para deixar o texto em negrito usamos a tag <strong>
pois ela e semantica, e significa que o texto marcado possui Força

Para O italico usamos a tag <em>
pois deixara semantico com o segnificado de enfase

Texto deletado e texto inserido

se quisermos usar um texto e dar sentido de que ele esta ali para ser lido,
porem deve ser desconsiderado, usamos a tag <del>,
mas se caso quisermos sublinar o texto para dar o sentido de que devemos dar atenção
usamos a tag <ins>

Texto sobrescrito e subscrito

Agora se quisermos falar de x2 e de h2o por exemplo
devemos usar as seguintes tags, para o numero ficar em cima usamos o <sup>
mas se quisermos adicionar um numero para baixo devemos usar o <sub>

codicos de outra linguagem de porgamação

quando queremos demonstrar ou adicionar um trecho de um codico
devemos usar a tag <code>
porem quando fazemos apenas isso, o codico fica meio estranho,
pois ele nao fica igual digitamos, para resolver isso e so usar a tag <pre>
que deixara o seu codico igual esta no visual code

Citações

Para criar citações devemos usar a tag <q>
do ingles quote, que singnifica citar, essa e mais usada quando
queremos usar a citação no meio do paragrafo

tambem podemos criar uma citação em bloco, usando a tag <blockquote>
servem para citações longas que precisam de um paragrafo so para si
se quisermos podemos colocar um link para a pagina que pegamos a citação
usando a tag <cite> dentro da tag <blockquote>

texto invertido
